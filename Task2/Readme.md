# Laundry Wallah Assignment Task 2

This project is a simple landing page for a laundry service website built with HTML.

## About the Page
The page in [index.html](index.html) introduces the business as "Laundry Wallah/Laundry Mart" and highlights the main services offered.

## Features Included
- Welcome section with a brief company introduction
- Services section listing laundry-related offerings
- Price list section showing sample service charges
- Booking form for users to submit their name, email, and place
- Contact information in the footer

## How to View
1. Open [index.html](index.html) in a browser.
2. You can also run a local server if you prefer.

## Purpose
This page is designed to provide a clean and attractive overview of the laundry service and encourage users to book their services.
