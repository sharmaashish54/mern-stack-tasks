# CSS Selectors Demo

This project demonstrates the use of CSS selectors in a simple HTML page.

## What this page includes
The page in [index.html](index.html) shows different styling examples using:
- Element selectors such as `p` and `li`
- Class selectors such as `.head`
- ID selectors such as `#bglime` and `#para`
- Attribute-based styling using `:nth-of-type()`
- Inline styling inside a span element

## Styling details
The styles are defined in [style.css](style.css) and include:
- A warm orange page background
- Blue heading styling for `.head`
- Lime background for the `#bglime` heading
- Red text for `.text`
- Green paragraph styling for `#para`
- Different link backgrounds for the first and second unordered lists

## How to view
1. Open [index.html](index.html) in your browser.
2. You will see a sample page that visually demonstrates how various CSS selectors work.

## Purpose
This task is intended to help learners understand how CSS selectors target and style HTML elements in different ways.
